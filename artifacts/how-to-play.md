# How to Play

## Join the Game

- Select "Join Game as Player" and enter the 6-digit code
- Set your display name
- Wait for the host to start the game

## Each Round

1. Listen as the host announces a number
2. Decide: Will the next number be **higher** or **lower**?
3. Submit your guess on your device (you can't change it once submitted!)
4. The host will announce the next number and reveal the answer
5. **Correct guess:** You stay in the game!
6. **Wrong guess or no guess:** You're eliminated (but keep playing for fun!)

## Winning

Be the last player standing to win!
